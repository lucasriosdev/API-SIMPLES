```bash
Node.js 18.x+
npm 9.x+ ou yarn 1.22.x+

bash
Copy

Instale as dependências:

bash
Copy
cd API-SIMPLES
npm install
⚡ Como Executar
Modo desenvolvimento (com hot-reload):

bash
Copy

bash
Copy

🌐 Endpoints
